

## Проблема

На дашборде две карточки показывают похожие числа без контекста:

- **Динамика**: `-1.2` за 3 месяца — разница биовозраста между двумя последними анализами
- **Прогресс**: `-4.2` лучше — разница биовозраста между ПЕРВЫМ и последним анализом

Пользователь не понимает разницу, потому что:
1. "Прогресс" не говорит "с какого момента" и "в чём измеряется"
2. Слово "лучше" ничего не объясняет
3. Оба числа выглядят одинаково — просто цифры без единиц

## Решение

Переименовать и добавить контекст обеим карточкам:

| Было | Стало |
|------|-------|
| **Динамика** `-1.2` за 3 месяца | **Последнее изменение** `-1.2 г.` за 3 месяца (между 2 последними) |
| **Прогресс** `-4.2` лучше | **Общий прогресс** `-4.2 г.` с первого анализа (дата) |

### Конкретные изменения в `src/pages/Dashboard.tsx`

1. **Карточка "Динамика"** (строки 579-598):
   - Заголовок: "Динамика" → "Последнее изменение"
   - Добавить единицу "г." после числа
   - Подпись: оставить период "за 3 месяца"

2. **Карточка "Прогресс"** (строки 600-622):
   - Заголовок: "Прогресс" → "Общий прогресс"
   - Добавить единицу "г." после числа
   - Подпись: "лучше"/"хуже" → "с {дата первого анализа}" (например "с янв. 2025")
   - Вычислить дату первого анализа из `sortedAnalyses[0]`

3. Добавить новую переменную `displayFirstAnalysisDate` для отображения даты первого анализа.
