

## Переливающийся логотип

Добавим CSS-анимацию `hue-rotate` на логотип, которая будет плавно менять тон (hue) по кругу, создавая эффект переливания.

### Изменения

1. **`src/index.css`** — добавить keyframe-анимацию `hue-shift` с переходом `filter: hue-rotate(0deg)` → `hue-rotate(360deg)` и утилитарный класс `.animate-hue-shift`.

2. **`src/components/landing/HeroSection.tsx`** — добавить класс `animate-hue-shift` на `<img>` логотипа.

3. **`src/pages/Auth.tsx`** — аналогично добавить класс на логотип.

4. **`src/components/AppSidebar.tsx`** — если там тоже есть логотип, добавить класс.

Анимация будет бесконечной, плавной (~4-6 секунд на цикл), без влияния на остальные элементы.
