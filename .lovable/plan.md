Но UI-лейблы действительно недостаточно ясны. Вот что нужно исправить:

---

## Проблема с UI

1. **Бейдж "Критическое"** — непонятно, что критическое. Критическое что? Влияние на что?
2. **Нет числового контекста** — убрали "X/10", но потеряли ощущение шкалы. Пользователь не понимает, насколько это серьёзно.
3. **Прогресс-бар без подписи** — полоска заполнена, но не ясно что она означает.

## Решение

В `src/components/risk-zones/AgingBlockers.tsx`:

- Бейдж: **"Критическое"** → **"Критическое влияние"** / **"Умеренное влияние"** / **"Слабое влияние"** — добавить слово "влияние" для контекста
- Под прогресс-баром добавить подпись: **"Влияние на старение: X/10"** мелким текстом — возвращаем числовую шкалу, но теперь с понятным контекстом
- Получается: бейдж даёт быстрое текстовое понимание, а подпись — точную числовую оценку

```text
┌─────────────────────────────────────────────┐
│ ⚠ Остаточное воспаление    [Умеренное влияние] │
│ ████████████████████░░░░░░░░░░              │
│                    Влияние на старение: 7/10 │
│                                             │
│ НА ОСНОВАНИИ ВАШИХ ДАННЫХ:                  │
│ • CRP 2.5 мг/л — ещё не оптимален (<1.0)   │
│ • Субклиническое воспаление ускоряет...     │
│                                             │
│ Рекомендация: Омега-3 до 3 г/день...        │
└─────────────────────────────────────────────┘
```
