

# Несоответствие биологического возраста между вкладками

## Проблема

Две вкладки дашборда показывают разные числа для одного и того же:

| Вкладка | Показывает | Формула | Результат |
|---------|-----------|---------|-----------|
| **Текущее состояние** | "−2.5 лет моложе" | `chrono_age_сейчас − bio_age` | 2.5 |
| **Динамика** (график, тултип) | "Разница: 2.9" | `bio_age − chrono_age_на_дату_анализа` | 2.9 |

Две причины расхождения:

1. **Разные знаки**: Dashboard считает `chrono − bio` (положительное = моложе), а BioAgeTrendChart считает `bio − chrono` (положительное = старше). Направление инвертировано.

2. **Разный хронологический возраст**: Dashboard берёт текущий возраст (`calculateAge(birth_date)` — возраст сегодня), а график пересчитывает возраст на дату каждого анализа. Если анализ был, например, 5 месяцев назад, хронологический возраст был на ~0.4 года меньше — отсюда разница 2.5 vs 2.9.

## Решение

Унифицировать формулу и отображение:

### 1. `BioAgeTrendChart.tsx` — инвертировать знак разницы

Строка 46: изменить `analysis.biological_age - chronologicalAge` на `chronologicalAge - analysis.biological_age`, чтобы положительное число = моложе (как на дашборде).

### 2. `BioAgeTrendChart.tsx` — обновить тултип

Строка 64-68: изменить логику цветов — теперь `difference > 0` = зелёный (моложе), `difference < 0` = красный (старше). Обновить текст: "моложе" / "старше" вместо просто "Разница".

### 3. Dashboard — пересчитать `ageDifference` на дату последнего анализа

Строка 402: вместо `calculateAge(birth_date)` (текущий возраст) использовать хронологический возраст на дату последнего анализа, чтобы число совпадало с графиком. Формула:
```
chronoAgeAtAnalysis = (analysisDate - birthDate) / 365.25 days
ageDifference = chronoAgeAtAnalysis - biologicalAge
```

Это гарантирует что "−2.9 лет моложе" на вкладке "Текущее" совпадёт с "Разница: +2.9 моложе" в тултипе графика.

### Конкретные файлы

**`src/components/dashboard/BioAgeTrendChart.tsx`**:
- Строка 46: `difference = chronologicalAge - analysis.biological_age` (было наоборот)
- Строки 64-68: `diffColor = data.difference > 0 ? 'text-green-600' : 'text-red-600'` (инвертировать)
- Строка 67: `diffSign = data.difference > 0 ? '+' : ''` → отображать "моложе"/"старше"

**`src/pages/Dashboard.tsx`**:
- Строки 401-402: вычислять `chronologicalAge` на дату последнего анализа, а не текущую дату
- Нужно найти дату последнего анализа из `allAnalyses` / `demoData.analyses`
