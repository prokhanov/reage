
## Что делаем

Превращаем «Стратегию здоровья» из списка задач в **аналитический раздел-прогноз**, который отвечает на вопрос «куда идёт моё здоровье и что будет, если ничего не менять». Тактика «что принимать сегодня» остаётся в Рекомендациях.

Выбраны направления: **A** (аналитика без задач), **B** (дорожная карта на год), **C** (одна главная цель квартала), **F** (прогноз с планом vs без плана).

## Новая структура страницы `/health-strategy`

```text
1. Заголовок + краткое объяснение «чем это отличается от Рекомендаций»
2. [НОВОЕ] Прогноз: с планом vs без плана  ← главный вау-блок
3. [НОВОЕ] Главная цель квартала            ← один фокус на 3 месяца
4. [НОВОЕ] Дорожная карта на год            ← вехи 1/3/6/12 мес
5. Карта рисков по системам (RiskMap)       ← остаётся, без изменений
6. Что мешает молодеть (AgingBlockers)      ← остаётся, без изменений
7. SmartPriorities                          ← УДАЛЯЕМ со страницы
```

SmartPriorities полностью убираем со «Стратегии». Конкретные задачи («принимать B12», «сдать ферритин») живут в `/recommendations` и `/prescriptions`.

## Новые компоненты

### 1. `ForecastComparison.tsx` — «С планом vs Без плана»

Две колонки рядом, визуальное сравнение через 12 месяцев:

```text
┌──────────────────────┬──────────────────────┐
│  БЕЗ ПЛАНА           │  С ПЛАНОМ            │
│  Био-возраст: 42 →47 │  Био-возраст: 42 →39 │
│  ▲ +5 лет старения   │  ▼ −3 года           │
│  Риск ССЗ:    ▲▲▲    │  Риск ССЗ:    ▼      │
│  Энергия:     низкая │  Энергия:     высокая│
│  3 системы в зоне 🔴 │  0 систем в зоне 🔴  │
└──────────────────────┴──────────────────────┘
```

Источник данных: текущий `risk_zone_analyses` + текущий био-возраст. Прогноз «без плана» = экстраполяция текущего тренда (используем `trend` из RiskMap). Прогноз «с планом» = целевые значения из существующих `prediction.improvement` агрегированно.

### 2. `QuarterGoal.tsx` — «Главная цель на 3 месяца»

Одна большая карточка с фокусом квартала:

```text
🎯 ФОКУС БЛИЖАЙШИХ 3 МЕСЯЦЕВ
Восстановить запасы железа

Почему именно это: ферритин 18 нг/мл — это причина
усталости, выпадения волос и низкой энергии.

Целевой показатель: ферритин 70+ нг/мл
Прогресс: ████░░░░░░ 18 → 70

Что это даст:
• Уйдёт хроническая усталость
• Восстановится концентрация
• Био-возраст −1.5 года

→ Конкретные шаги — в разделе «Рекомендации»
```

Логика выбора цели: берём систему с самым высоким `risk_score` и худшим `trend`, вытаскиваем главный лимитирующий биомаркер.

### 3. `YearRoadmap.tsx` — «Дорожная карта на год»

Горизонтальный таймлайн с 4 вехами:

```text
Сейчас ──●──── 1 мес ──●──── 3 мес ──●──── 6 мес ──●──── 12 мес
         │             │             │             │
         Старт         Первые        Главная       Цель года
         замеров       сдвиги        цель          достигнута
                       биомаркеров   квартала      
                       (B12, D)      (ферритин)    Био-возраст:
                                                   −3 года
```

Каждая веха — кликабельная точка с поповером: что должно произойти, какие биомаркеры подтянутся, какие пересдать.

## Уточнение позиционирования

В шапке страницы добавляем info-блок, объясняющий разницу:

> **Стратегия** — это карта вашего здоровья на год: куда вы идёте, какая главная цель и что будет, если ничего не менять.
> **Рекомендации** — конкретные шаги (что принимать, какие анализы сдать), чтобы пройти эту карту.
