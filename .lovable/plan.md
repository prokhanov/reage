
# План: Объёмная 3D-лента в Hero секции

## Что нужно сделать

Полностью переделать компонент `GradientRibbon.tsx`, чтобы создать реалистичную объёмную 3D-ленту как в референсе.

## Ключевые изменения

### 1. Новая геометрия ленты
- Изменить путь SVG так, чтобы лента проходила **по центру экрана** волной (S-образная кривая)
- Лента должна начинаться слева снизу, подниматься к центру, затем опускаться вправо вниз

### 2. Техника создания объёма
Вместо stroke-линий использовать **заливку (fill)** с несколькими слоями:

- **Основное тело ленты** - замкнутый путь (не stroke, а fill) с градиентом pink-to-purple
- **Радиальный градиент для объёма** - имитация цилиндрической формы с более светлым центром и тёмными краями
- **Верхний блик** - яркая полоса вдоль верхнего края для эффекта отражения света
- **Нижняя тень** - тёмная область снизу для глубины
- **Рассеянное свечение** - glow-эффект под лентой

### 3. Градиенты и эффекты
- Основной градиент: ярко-розовый (#ff1493) → фуксия (#ff69b4) → орхидея (#da70d6) → фиолетовый (#9370db) → синий (#6366f1)
- Перпендикулярный градиент для 3D-объёма (светлый центр, тёмные края)
- Gaussian blur для свечения под лентой

### 4. Позиционирование
- Центрировать SVG так, чтобы волна проходила через середину viewport
- Убрать агрессивные отступы `-top-[40%]`

## Технические детали

```
Файл: src/components/landing/GradientRibbon.tsx

Изменения:
1. SVG viewBox и позиционирование - центрирование ленты
2. Новые пути для волнообразной формы по центру
3. Использование fill вместо stroke для основного тела
4. Добавление радиального градиента для цилиндрического объёма
5. Слои: glow → main body → highlight → shine line
```

## Результат

Лента будет выглядеть как настоящий 3D-объект - объёмная труба с реалистичным освещением, проходящая волной через центр Hero-секции, как в референсе.
