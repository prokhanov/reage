

# Дифференциация структуры без статичных метафор

## Проблема пользователя

Статичные метафоры ("тлеющий костёр") повторятся в каждом из 4 отчётов в год. Нужна **вариативность по инструкции**, а не по тексту.

## Решение

Вместо прописывания конкретных фраз — дать AI **тип приёма** и **запрет на повторение**. Каждый раздел получит уникальную структуру, но без захардкоженных метафор.

### Что меняем (6 UPDATE к `ai_prompt_settings`)

**1. Заголовки секций — уникальные для каждой системы** (anchor теги не меняются):

| Система | ⏳ Старение → | 🔍 Особенности → |
|---|---|---|
| ⚡ Энергия | Что истощает энергетический ресурс | Ваш энергетический профиль |
| ❤️ Сердце | Что повышает кардиометаболический риск | Сосудистый профиль |
| 🛡 Воспаление | Что поддерживает хроническое воспаление | Характер иммунного ответа |
| 🧬 Эндокринная | Что нарушает гормональный баланс | Гормональный профиль |
| 🔄 Метаболизм | Что перегружает метаболизм | Метаболический паспорт |
| 💧 Почки | Что увеличивает нагрузку на почки | Водно-солевой баланс |

**2. Вариация количества элементов** — разные числа для разных систем (3-5 инсайтов в одной, 2-3 в другой), чтобы ритм чтения менялся.

**3. Уникальные акценты в рекомендациях** — вместо одинакового "питание, сон, нутрицевтики" для всех:
- Почки: водный режим, минеральные воды, ограничение соли
- Воспаление: противовоспалительная диета, специи (куркумин), омега-3 протоколы
- Эндокринная: циркадные ритмы, адаптогены, гормон-поддерживающие протоколы
- и т.д.

**4. Динамический крючок вместо статичной метафоры** — в начале резюме каждой системы добавляем инструкцию:

```
Начни резюме с яркого, нешаблонного вступления,
которое отражает КОНКРЕТНУЮ ситуацию пациента.
Не используй заученные метафоры.
Каждый раз формулируй вступление заново,
исходя из данных.
Варьируй стиль: вопрос, аналогия,
неожиданный факт, прямой вывод.
```

Это гарантирует, что AI каждый раз придумывает новое вступление на основе данных пациента, а не повторяет одну и ту же фразу.

**5. Сокращение «Роль системы»** — вместо 4 пунктов теории → инструкция "1 абзац (3-4 предложения) о роли системы В КОНТЕКСТЕ данных пациента, без общей теории".

### Объём работ
- 6 UPDATE запросов к `ai_prompt_settings` (`category_*_user`)
- Только тексты промптов, код не меняется
- Якорные теги остаются прежними
