## Что делаем

Удалить с публичной страницы «Подготовка к анализам» (/prep) блок «Особенности по типам анализов», который содержит 4 карточки: щитовидная железа, глюкоза/инсулин, липидный профиль, витамины и минералы.

## Файл для изменений

- `src/pages/AnalysisPrep.tsx`

## Изменения

1. Удалить секцию «Особенности по типам анализов» целиком (разметку section, заголовок, grid с карточками).
2. Удалить неиспользуемую константу `SPECIAL`, которая питала этот блок.
3. Удалить неиспользуемый импорт `Activity` из `lucide-react`.

## Что не трогаем

- Остальные блоки страницы: «Общие правила», «Сбор мочи» и другие секции остаются без изменений.
- Страница продолжит открываться по тому же адресу /prep.

## Проверка после внедрения

- Сборка проекта не падает.

- В консоли браузера нет ошибок, связанных с удалённым импортом или константой.
