
# План: Расширение Hero-секции — полный спектр возможностей

## Проблема
Текущий hero фокусируется только на биологическом возрасте, игнорируя другие ключевые ценности продукта:
- Отслеживание трендов здоровья во времени
- AI-анализ и персональные рекомендации
- Мониторинг 5 систем организма
- Динамика изменений и прогресс

## Решение

### 1. Обновить главный заголовок
**Было:**
```
Ваши анализы скажут,
сколько вам на самом деле
```

**Станет:**
```
Ваше здоровье в цифрах,
динамике и рекомендациях
```

*Альтернативный вариант:*
```
Узнайте свой биологический возраст.
Отслеживайте тренды. Улучшайте здоровье.
```

### 2. Расширить подзаголовок
**Было:**
```
Один забор крови. Точный расчёт биологического возраста.
Медсестра приедет к вам.
```

**Станет:**
```
50 биомаркеров → биологический возраст → тренды по системам → AI-рекомендации.
Медсестра приедет к вам. Результат за 5 дней.
```

### 3. Обновить badge
**Было:**
```
50+ биомаркеров • AI-анализ • Забор на дому
```

**Станет:**
```
Биовозраст • Тренды здоровья • AI-рекомендации • Забор на дому
```

### 4. Переработать StatCards — показать полный цикл ценности

**Новые 4 карточки (вместо 3):**

| Иконка | Значение | Подпись |
|--------|----------|---------|
| Activity | 50+ | биомаркеров |
| TrendingUp | 5 систем | под контролем |
| Brain | AI | персональный анализ |
| Calendar | 4× в год | отслеживание трендов |

### 5. Изменить CTA
**Было:** "Узнать свой возраст"
**Станет:** "Начать мониторинг" или "Узнать своё здоровье"
