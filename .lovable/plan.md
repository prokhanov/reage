

# Перенос захардкоженных инструкций в редактируемые промпты

## Проблема

В edge function `analyze-biomarkers` есть два блока текста, которые добавляются программно и **не видны/не редактируемы** в разделе AI Settings:

1. **Глобальный контекст биомаркеров** (строка 651) — добавляется к user prompt каждой категории: список всех сданных маркеров + инструкция "не рекомендуй сдавать то, что уже есть"
2. **Инструкция согласованности** (строка 826) — добавляется к system prompt резюме: "не противоречь детальным разделам"

## Решение

### 1. Добавить плейсхолдер `{globalBiomarkers}` в user промпты категорий

- В edge function: вместо конкатенации `+ "\n\n" + globalBiomarkersContext` использовать `.replace(/{globalBiomarkers}/g, globalBiomarkersContext)`
- Если плейсхолдер не найден в промпте — fallback на конкатенацию (обратная совместимость)
- Обновить все `category_*_user` промпты в БД: добавить блок с `{globalBiomarkers}` и инструкцией

### 2. Перенести инструкцию согласованности в `summary_system` промпт в БД

- Убрать хардкод на строке 826 (конкатенация `+ "\n\nВАЖНО: Твоё резюме ДОЛЖНО..."`)
- Обновить промпт `summary_system` в БД — добавить этот текст в конец

### Изменения в файлах

**`supabase/functions/analyze-biomarkers/index.ts`**:
- Строка 644-651: заменить конкатенацию на `replace` с fallback
- Строка 826: убрать хардкоженную конкатенацию, передавать `summarySystemPrompt` as is

**БД (`ai_prompt_settings`)**: обновить промпты через insert tool:
- Все `category_*_user` промпты: добавить `{globalBiomarkers}` в конец
- `summary_system`: добавить инструкцию согласованности в конец
